./ds_part contains the statistics of defense systems identified in each strain, including their abundance, types, and subtypes.
./prophage_part contains the list of prophages identified in Pseudomonas aeruginosa, together with the defense systems and anti-defense genes encoded within each prophage.
strains_info contains the metadata of all strains, including their geographical origins, collection dates, and sequence types (STs).
enrich_data contains the results of the enrichment analyses.
